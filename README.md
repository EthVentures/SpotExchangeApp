# Prerequisites
* Docker Compose

# Configuration

Rename default.env to .env and edit where necessary.
```bash
mv default.env .env
```


# Launching Application

To launch natively:
```bash
ionic serve
```

To launch using docker:
```bash
docker-compose build && docker-compose up
```
